# Tools for geometry

## STL to UDC converter

Converter converts STL to UDC and adds additional data to each vertex  
with default temperature 298.15 K.
```bash
python stlToUdcConverter.py inputStl.stl outputUdc.inp
```  
Where `inputStl.stl` is STL input file  
Where `outputUdc.inp` is UDC output file
